GIT

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.

  • sistema de controlo de versões
  • gratuito
  • open source
  • rápido
  • sistema distribuído, ou seja, cada utilizador pode ter uma cópia do repositório localmente!
  • é o mais utilizado

comando para ver se temos instalado e qual a versão: git --version

para iniciar um repositório local: git init

para consultar o status dos ficheiros: git status

para adicionar ficheiro ao staging area / index / área de preparação: git add intro_git.html

git add ficheiro1 ficheiro2

git add .

quando for para guardar uma versão do programa / fazer commit / efetuar snapshot: git commit -m "pequena descrição"


para ver o histórico de commits: git log

git log --oneline --graph

ramificação do código

listar os ramos/branch e indicar em qual estamos git branch

criar uma nova ramificação / branch git branch nomebranch

para trocar de ramo: git switch nomebranch

para efetuar o merge git merge nomebranch

outros comandos

para remover um ficheiro do staging area (o contrário de git add nomeficheiro): git restore --staged nomeficheiro

para ver diferenças entre versões: git diff
